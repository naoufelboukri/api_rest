Module WEB_4 : Etape 5

Consignes

Le rendu de cette étape se fera dans un dossier nommé Etape_5. L'application communiquera avec votre API rendue à l'étape 4.

Vous devez mettre en place une application ReactJS ou Angular qui communique avec votre API. Voici les fonctionnalités obligatoires :

  • L'utilisateur peut s'enregistrer et se connecter.
  • La connexion est obligatoire pour accéder au reste de l'application.
  • On doit voir la liste des utilisateurs.
  • En cliquant sur un utilisateur, l'application affiche ses informations et ses adresses.
  • Un utilisateur normal peut modifier ou supprimer son profil.
  • Un administrateur peut modifier et supprimer les utilisateurs.
  • Un utilisateur a le contrôle sur ses adresses (ajout / modification / suppression).
  • Un administrateur a le contrôle sur toutes les adresses (ajout / modification / suppression)

Vous pouvez fournir une version compilée de votre application ou bien une version à lancer avec NodeJS, auquel cas vous devez fournir un README pour expliquer comment lancer votre application.
